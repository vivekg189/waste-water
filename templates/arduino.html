Firmware format

Send one line per reading with Serial.println(). Examples this server understands:

  • 42.5 or LEVEL:42.5
  • {"level":18.2} (JSON with level, value, distance, …)

Change port

Default is COM9. To use another port, set environment variables before starting Flask:

set ARDUINO_PORT=COM3
set ARDUINO_BAUD=115200
